Aristotle Discovery Layer

(2011-)

#}
  • Started as a fork of the Django-based Kochief (http://code.google.com/p/kochief/)
  • Indexes MARC21 records exported from legacy catalog into Apache Solr (https://lucene.apache.org/solr/)
  • Runs on a Ubuntu server VM on Colorado College's VMWare cluster
  • The pymarc Python module critical to parsing/processing
  • Repository available at https://github.com/jermnelson/Discover-Aristotle


  • Based on Django
  • API interface to Redis Datastore using redis-py module
  • Fedora commons digital repository utilities created in Python with Django and Bootstrap for the front-end
  • Source code for the Redis Library Services Platform is in two open-source projects; the
    • Aristotle Library Apps, source code available at https://github.com/jermnelson/aristotle-library-apps
    • BIBFRAME Datastore, source code availlable at https://github.com/jermnelson/BIBFRAME-Datastore
