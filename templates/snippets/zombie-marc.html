Persistence of MARC21

MARC (an acronym for MAchine-Readable Cataloging) is a data format for bibliographic, authority, and holdings data. The Library of Congress is responsible for initially creating MARC in the 1960s and continues to support to the current time. The MARC format combines both fixed control fields and variable length fields with a number of different subfields. Most library catalogs use MARC format, either using the format natively or converting records encoded in MARC to relational database or other backend storage. Colorado College's Tutt Library has a collection of over 1,030,000 MARC21 records representing Colorado College's physical and online collections.
