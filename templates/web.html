PYCON 2014 Poster

Replacing a Legacy Library Catalog @ Colorado College

This companion webpage accompanies a poster session by Jeremy Nelson at the annual Python programming conference, PYCON 2014.

Web Application Hyperlinks

  • http://discovery.coloradocollege.edu/ Existing Django Discovery Layer for Colorado College, licensed under Apache 2
  • http://tuttdemo.coloradocollege.edu/ - Redis Demostration Discovery Layer using MARC21 records from Prospector and Colorado College's digital archives, licensed under Apache2
  • http://catalog.coloradocollege.edu/ - First MVP of new catalog for Colorado College using the Catalog Pull Platform, licensed under th GPLv2.

Source Code Repositories

  • Discovery Aristotle https://github.com/jermnelson/Discover-Aristotle - Forked from Kochief, uses Solr for search along with Django for the web-framework.
  • Aristotle Library Apps href="https://github.com/jermnelson/aristotle-library-apps - Django HTML5 mobile and tablet apps for discovering, accessing, and managing library information. Bibliographic records along with other library and organizational specific informations are linked, stored, and managed using Redis.
  • BIBFRAME-Datastore https://github.com/jermnelson/BIBFRAME-Datastore, a Redis-based project for managing bibliographic records and operational data focused on libraries and other similar information-based organization
  • Flask-BIBFRAME - extension provides Linked Data support for the BIBFRAME (http://bibframe.org/) model and vocabulary. Code at https://github.com/jermnelson/flask_bibframe
  • Flask-SchemaOrg - extension for supporting the http://schema.org Linked Data vocabulary. Source code at https://github.com/jermnelson/flask_schemaorg
  • Flask-FedoraCommons - extension provides an interface to a Fedora Commons digital repository. Code available at https://github.com/jermnelson/flask-fedora-commons.
